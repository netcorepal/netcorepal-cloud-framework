    关于AI工具¶

    基于本项目采用的代码组织方式，每个类型例如领域模型、命令处理器、事件处理器等等都有明确的职责和能力，这对AI工具的辅助使用非常友好，AI工具可以根据你编写代码的文件模型和上下文准确推断出意图并给予代码补全提示，并且随着你的项目代码的增多，AI工具的智能程度会越来越高。

    因此，非常推荐使用AI工具来辅助编写代码，这样可以大大提高编码效率。

    一些技巧¶

    下面一些技巧可以帮助AI工具更好地辅助你编写代码：

    • 使用带有业务含义的类型命名
    • 使用带有明确功能含义的方法命名
    • 适当的注释

    AI工具推荐¶

    推荐使用 GitHub Copilot 来作为代码辅助工具，它支持多种编辑器，例如Visual Studio Code、Visual Studio、JetBrains Rider等等。
